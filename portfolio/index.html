    I'm a Full Stack Software Engineer with expertise in Java, Python, React.js, and C/C++, specializing in building scalable, data-driven applications from concept to cloud deployment. With a Master's degree in Computer Science from the University at Buffalo and over three years of hands-on experience across startup and enterprise environments, I bring end-to-end expertise in full-stack development, cloud architecture, and modern DevOps practices.


    I have experience building backend services with Java and Python along with developing frontend interfaces using React and TypeScript, which allows me to approach problems with a clear end to end understanding. As a founding engineer at Trulogik, I designed the API layer for a healthcare claims dashboard using Spring Boot to enable reliable and real time claims processing, and paired that with developing interactive React UI components that consumed those APIs. I also built a Python FastAPI based REST backend, deployed it on AWS Lambda, and set up a GitHub Actions pipeline that dockerized the service, pushed images to ECR, and integrated the system with RDS and S3. I rely on object oriented principles and common design patterns to keep code modular and scalable, and these industry-standard practices are fundamental to engineering excellence at leading tech companies.


    I'm actively seeking opportunities in software development roles where I can leverage my expertise in cloud-native architecture, full-stack development, and modern deployment workflows to contribute to innovative, high-impact projects.
